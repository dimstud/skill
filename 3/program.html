Как работать на курсе

  1. Изучить теги
  2. Попрактиковаться
  3. Выполнить домашнее задание

Описание курса

В первой части курса вы познакомитесь сщ стандартами Web 2.0, научитесь работать с макетами и форматировать код, освоите адаптивную верстку, а по итогу - сможете создавать быстрые и удобные сайты, котрорые точно понравяться пользователям.

Вы поймете название основных тегов HTML, сможете без усилий стилизовать и размещать элементы с помощью CSS и сверстаете свой первый макет.

Вторая часть курса сделат из вас настоящего мастера. Вы изучите сборщики проектов, научитесь верстать email-рассылки, разберетесь с анимацией на сайте.

Помимо этого, вы сверсаете фрагмент из интернет-магазина - главную страницу, страницу каталога и страницу товара!

Программа курса

  • Введение
  • Базовый HTML
  • Базовый CSS
  • Работа с макетом
  • Layout. HTML-разметка
  • Layout. Flexbox
  • Layout. Стилизация
  • Продвинутый HTML. Формы
  • Продвинутый CSS
  • Сетки
  • Адаптивность. Десктоп и планшет
  • Адаптивность. Мобильные устройства
  • JavaScript для верстальщика
  • Доступность
  • Кроссбраузерность
